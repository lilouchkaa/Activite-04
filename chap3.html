Partie 03: Résumé chapitre 03


1)Les Composants matériel d’un Réseau informatique:

Les cartes réseau :

  • Appelées cartes d'interface réseau, sont des composants matériels qui permettent à un ordinateur de se connecter à un réseau, comme Internet ou un réseau local.

    • Switch:

  • Il permet à plusieurs appareils de communiquer entre eux en utilisant des adresses MAC.Ils sont disponibles en différentes tailles et peuvent également offrir des fonctionnalités avancées.

    • HUB:

  • C'est un composant matériel utilisé pour connecter plusieurs appareils sur un réseau local.Il fonctionne comme un répéteur, il reçoit les données envoyées par un appareil et les transmet à tous les autres appareils connectés.

    • 2)Le Modèle OSI:

    C'est un modèle de référence théorique pour les communications de réseau et n'est pas utilisé dans les protocoles de communication de réseau modernes tels que TCP/I,il divise les communications en sept couches distinctes:

  • La couche physique: responsable de la transmission des données sur le support physique.
  • La couche liaison de données: responsable de la transmission des données entre les appareils qui partagent un même support physique
  • La couche réseau : responsable du routage des données à travers le réseau.
  • La couche transport: responsable de l'acheminement des données de bout en bout entre les appareils.
  • La couche session : responsable de l'établissement et de la gestion des sessions entre les appareils.
  • La couche présentation : responsable de la conversion des données dans un format qui peut être compris par l'appareil de destination.
  • La couche application : responsable de la communication entre les applications sur les appareils de réseau.

    • 3) Le Modèle TCP/IP (Transmission Control Protocol/Internet Protocol)

    C’est une référence pour les communications de réseau, visant a créés et mettre en œuvre des réseaux informatiques modernes .composer de 4 couches de communications principales :

  • La couche liaison de données : assure le transfert des données sur supports physiques.
  • La couche réseau : chargée de déterminer le chemin des données dans le réseau.
  • La couche transport : se charge de l'acheminement des données entre les appareils.
  • La couche application : facilite de la communication entre les applications sur les périphériques réseau.

    • 4) L’adressage

    Les règles de communications dans un réseau :

  • Adressage : identifiant de manière unique deux entités communicantes
  • Architecture : définis les rôles des deux unités

    • Une adresse IP (Internet Protocol) est un numéro identifiant chaque ordinateur sur un réseau auquel il appartient.
    Il y’a deux mode :

  • adresse IPV4: (32 bits) Chaque paquet contient : adresse source et adresse destinataire
  • adresse IPV6 : (128 bits) Le texte est représenté en divisant le mot en 8 mots de 16 bits, séparés par le caractère ' : '.

    • 5) L’adresse IPV4

    -Chaque adresse IPV4 (4 octets) contient deux parties :

  • Adresse réseau : des bits de valeur supérieure
  • Adresse hôte : les bits déterminent le nombre des machines possible sur ce réseau

    • -(Bits partie réseau) = (bits partie hôte) alors le réseau est un {hôte}

    -Les types d’adresse IPV4:

  • L’adresse réseau : les bits partie hôtes=0 donc les machines du réseau ont la même partie réseau
  • L’adresse de diffusion : les bits de partie hôtes=1
  • L’adresse hôtes : attribue les valeurs entre l’adresse réseau et l’adresse de diffusion sur {machines} dans ce réseau.
  • Le masque de réseau et de sous réseaux : Tous les bits de la partie réseaux = 1
